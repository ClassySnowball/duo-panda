Planning a flashcard PWA for Dutch, Polish, and English learning.

- **Tech**: Next.js PWA deployed on Vercel, Supabase for DB + auth
- **Content**: Pre-built decks + user-created cards, text-only for now
- **Algorithm**: Spaced repetition (SM-2 or similar)
- **Card types**: Words, sentences, phrases — all three languages interconnected (NL↔PL↔EN)
- **Categories**: Food, travel, relationships, hiking, dogs
- **Gamification**: Daily goals (cards reviewed + new words learned), streak tracking, progress stats
- **Data**: Supabase Postgres (persistent, cloud-synced across devices)
- **Auth**: Supabase Auth (simple login)
- **Content sources**: AI-generated examples + free/open-source vocab lists (MUSE, Tatoeba, FrequencyWords, Anki decks)
- **Import**: User will add custom word lists later
- **Design**: Outdoor/trail-inspired, earth tones (browns, greens, rust), journey metaphor — discovering words like exploring trails
- **Development**: Claude generates full codebase, friend helps only if needed

**Why:** Davey is actively learning both languages and wants a personalized tool.
**How to apply:** Keep the app simple and non-technical to set up. Prioritize usability over developer complexity. Generate complete, deployable code.
